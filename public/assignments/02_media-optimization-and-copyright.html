02 Media Optimization and Copyright

This week I learned some about media optimization...

Image Optimization

Optimized image parameters:

Image size (longest edge)
1920px
Image compression
65%
File size
100.69KB

Use any raster image editor and save/export image as jpg or webm

Video Optimization

Optimized video parameters:

Video length
10 seconds
Frame rate
25fps (frames per second)
Video size
1280x720px (720p, HD)
File size
less than 2MB

ffmpeg command line:

ffmpeg -i input-video.mov -ss 00:00:10 -to 00:00:17 -vf "scale=1280:720,setsar=1:1, fps=25" -c:v libx264 -crf 30 -preset slow -an video.mp4
